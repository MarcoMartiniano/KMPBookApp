# KMP Book App 📚

A study project built with Kotlin Multiplatform (KMP) and Compose Multiplatform.

The app consumes a Book API and demonstrates modern Android/iOS/Desktop development using:
- Compose Multiplatform
- Clean Architecture
- MVI
- Dependency Injection (Koin)
- Kotlin Coroutines
- Shared business logic

## Learning Purpose

This repository was created for educational purposes while following the excellent Compose Multiplatform Crash Course by Philipp Lackner.

Original video:
https://www.youtube.com/watch?v=WT9-4DXUqsM

The implementation closely follows the tutorial as part of my Kotlin Multiplatform learning journey.
